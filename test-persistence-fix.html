Test Text Style Persistence Fix

This page tests whether the text styling persistence fix works

Instructions:

  1. Open SVGEdit
  2. Click the text tool and create some text
  3. Use TopPanel controls to change text alignment (e.g., left align)
  4. Note the visual alignment
  5. Refresh the browser page
  6. Check if the text alignment is preserved after refresh

Expected Behavior with Fix:

  • Text alignment changes should be immediately visible
  • Text alignment should persist after browser refresh
  • No need to interact with TopPanel after refresh to see correct alignment

Technical Details:

The fix adds a restoreTextDivStyles() function that runs after SVG loading to ensure that XHTML div styles are properly restored after XML serialization/deserialization.

The function re-applies the CSS styles from the style attribute to ensure the CSS object model is properly initialized.
