Text Alignment Test

Manual Test Instructions:

  1. Open SVGEdit in the iframe below
  2. Click the Text tool (T icon)
  3. Click on the canvas to create a text element
  4. Type some text (e.g., "Test Text")
  5. Make sure the text element is selected
  6. Try changing the text alignment using the alignment dropdown
  7. Watch for any modals saying "Changes you made cannot be saved"
